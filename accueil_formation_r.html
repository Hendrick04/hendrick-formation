À propos de la formation

Face à l'évolution croissante du volume des données dans les organisations, la gestion et l'exploitation des données deviennent un défi majeur. La donnée est un actif important accélérant le développement de toute entreprise ambitieuse.

Cette formation, taillée sur mesure par des professionnels et experts de la data, vous donnera les outils pour gérer et exploiter vos données sereinement — de l'architecture d'une Data Office jusqu'au reporting Power BI, en passant par l'ingénierie de données (Talend, SQL, Python), la modélisation Data Vault et l'automatisation avec n8n.

Public cible

Data Engineers Data Analysts Chefs de projet BI Toute personne gestionnaire de données

Prérequis

Bonne connaissance d'Excel Aisance avec les chiffres R ou Python (un plus)

Objectifs de la formation

  • Comprendre les composantes d'une Data Office moderne : architecture, gouvernance, engineering, BI, monitoring
  • Concevoir des flux ETL/ELT avec Talend et manipuler des données avec SQL et Python/Pandas
  • Modéliser un entrepôt de données robuste avec la méthode Data Vault 2.0
  • Créer des rapports dynamiques et maîtriser le langage DAX dans Power BI
  • Orchestrer et automatiser des pipelines de données de bout en bout avec n8n
  • Mener un projet individuel complet, de la donnée brute au tableau de bord narratif présenté à l'oral
Ta progression 0/6 modules validés

Programme — 6 modules

01
Introduction à la Data Office
Architecture · Gouvernance · Engineering · BI · Monitoring
8h Disponible Commencer
Objectifs
  • Comprendre les composantes d'une Data Office moderne et leur liaison
  • Se familiariser avec les 5 piliers : architecture, gouvernance, engineering, BI, monitoring
Outils abordés
Power BI Tableau n8n
Contenu
  • Entrepôt de données — Data Lake, DWH, DataMart : rôles et différences
  • Gouvernance de la donnée — qualité, fiabilité, sécurité, conformité
  • Engineering — flux ETL/ELT, orchestration avec n8n, pratiques DataOps
  • Analyse et Visualisation — outils Power BI / Tableau, construction de KPIs
  • Monitoring des données — logs, alerting, observabilité

Cas d'usage : Design d'une Data Office de bout en bout dans une Banque

02
Collecte et Manipulation des Données
Talend · SQL · Python / Pandas
15h Disponible Commencer
Objectifs
  • Comprendre les composants de Talend et concevoir des flux ETL
  • Manipuler et interroger des données structurées en base de données
  • Charger, nettoyer et transformer des données avec Python pour l'analyse
  • Collecter des données multi-sources et les charger dans une base
Partie 1 — Talend
  • Interface, projets, jobs, métadonnées de Talend Open Studio
  • Connexion CSV, PostgreSQL et API REST (tRestClient)
  • Transformations : tFilterRow, tMap, tReplace
  • Chargement PostgreSQL via tOutput · Monitoring de jobs
Partie 2 — SQL
  • SELECT, WHERE, ORDER BY, DISTINCT · COUNT, SUM, AVG
  • Jointures INNER, LEFT, RIGHT, FULL sur tables ventes/clients
  • NULL, COALESCE, CASE WHEN, GROUP BY HAVING
  • Création de vues et procédures stockées
Partie 3 — Python / Pandas
  • Chargement : read_csv(), read_json(), read_sql() · Connexion PostgreSQL
  • Nettoyage : doublons, valeurs manquantes, changement de types, renommage
  • Transformations : filtres, groupby, agrégations, colonnes calculées, jointures de DataFrames
  • Export vers CSV, Excel ou PostgreSQL

Cas d'usage : Pipeline d'ingestion CSV + API → PostgreSQL · Nettoyage de données brutes (doublons, types, formats)

03
Modélisation et Stockage avec Data Vault
SQL · PostgreSQL · Data Vault 2.0
15h Disponible Commencer
Objectif
  • Concevoir un entrepôt de données modulaire avec la méthode Data Vault 2.0
Théorie
  • Hubs, Links, Satellites — rôles et structure
  • Raw Vault vs Business Vault
  • Avantages Data Vault vs modèle en étoile
Pratique SQL
  • Création de tables Hubs, Links, Satellites en SQL
  • Chargements par étapes : staging, raw, business
  • Historisation et gestion des clés

Cas d'usage : Construction d'un mini-DWH en Data Vault à partir de données clients & ventes

04
Visualisation et Reporting avec Power BI
Power BI · DAX · Storytelling
10h Disponible Commencer
Objectifs
  • Créer des rapports dynamiques et interactifs
  • Maîtriser les filtres, modèles et le langage DAX
Outil
Power BI Desktop DAX
Contenu
  • Connexions aux sources de données : CSV, PostgreSQL, Excel
  • Modèle de données, relations et formatage
  • Introduction au DAX : colonnes calculées et mesures — SUMX, CALCULATE, FILTER, YTD
  • Storytelling avec Power BI : tableaux de bord, visuels, slicers

Cas d'usage : Tableau de bord des ventes avec filtres temporels et indicateurs personnalisés · Rapport ALM (Asset Liability Management)

05
Industrialisation & Automatisation
Talend · N8N · Power BI Service
6h Disponible Commencer
Objectifs
  • Orchestrer des processus de bout en bout avec n8n
  • Mettre en place des traitements automatisés de données
  • Planifier et monitorer des rafraîchissements de rapports Power BI
Contenu
  • Introduction à l'automatisation avec n8n
  • Appel API pour l'extraction de données ouvertes ou internes
  • Exécution de scripts Python via n8n (Node Execute Command)
  • Déclenchement de traitements ETL ou nettoyage automatique
  • Connexion Power BI Service : rafraîchissement de dataset · Alertes e-mail / Teams
  • Publication automatique de rapports mis à jour

Workflow complet : Source → Traitement Python → Chargement PostgreSQL → Rafraîchissement Power BI → Notification email

06
Projet Individuel — Reporting Power BI
De la donnée brute à la visualisation interactive
Projet final Disponible Commencer
Objectifs
  • Appliquer l'ensemble des compétences acquises sur un jeu de données réel
  • De la donnée brute à la visualisation interactive — présentation orale incluse
Exigences du projet
  • Connexion à une ou plusieurs sources de données
  • Transformation des données
  • Modélisation relationnelle
  • DAX personnalisé — mesures et colonnes calculées
  • Tableau de bord narratif — storytelling visuel
  • Workflow automatisé
  • Présentation orale de 5 minutes

Livrable : Rapport Power BI complet — connexion, transformation, modèle, DAX, dashboard narratif, automatisation — présenté à l'oral
